##### Neofetch_Custom -- modified config scripts for neofetch
### Conf contains the custom neofetch configs
### Ascii contains the custom ascii art

##### USAGE 

``` neofetch --config <path/to/config.conf --ascii path/to/ascii/art.txt ```
